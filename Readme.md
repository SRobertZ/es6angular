# es6Angular
Приложение, демонстрирующее использование кода ES6 в рамках приложения AngularJs 1.x

#Setup
- `git clone git@github.com:SRobertZ/es6angular.git`
- `cd es6Angular`
- `npm install -g gulp`
- `npm install -g bower`
- `npm install`
- `bower install`

##Сборка

Run `gulp build`

##Запуск проекта
Run `gulp serve`
